# 📊 Импорт расчётов услуг в PhpMyAdmin

## Быстрый способ (рекомендуется)

1. **Откройте PhpMyAdmin** → выберите вашу базу данных
2. **Перейдите во вкладку "Импорт"** (Import)
3. **Выберите файл** `db/pricing_data.sql` из вашего компьютера
4. **Нажмите "Выполнить"** (Go)
5. ✅ Расчёты для всех услуг загружены!

## Если импорт не работает

**Способ 1: Через PhpMyAdmin SQL-редактор**
1. Откройте вкладку "SQL"
2. Скопируйте содержимое файла `db/pricing_data.sql`
3. Вставьте в текстовое поле
4. Нажмите "Выполнить"

**Способ 2: Через командную строку**
```bash
mysql -u username -p database_name < db/pricing_data.sql
```

## Что загружается?

- ✅ **Типография и полиграфия** (10 материалов)
  - Визитки, буклеты, бланки, календари
  
- ✅ **Сувенирная продукция** (9 материалов)
  - Кружки, футболки, ручки, подарочные наборы
  
- ✅ **Широкоформатная печать** (8 материалов)
  - Баннеры, наклейки, печать на холсте, стикеры
  
- ✅ **Наружная реклама** (8 материалов)
  - Световые короба, вывески, входные группы, брендирование

**Итого: 35 материалов для расчётов**

## Проверка после импорта

1. Откройте сайт → страницу "Услуги"
2. Выберите любую услугу (кнопка "Заказать")
3. Откроется форма с калькулятором 🧮
4. Меняйте параметры → цена обновляется в реальном времени
5. Отправьте заявку → в админ-панели видна сохранённая стоимость

## Редактирование расчётов позже

Если нужно изменить расчёты:
1. Откройте админ-панель
2. Перейдите в "Управление расчётами услуг"
3. Добавляйте, удаляйте или меняйте материалы
4. Сохраняйте изменения
