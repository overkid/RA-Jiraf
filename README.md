Краткий гайд по запуску проекта локально.

## Стек
- **PHP** (страницы + API)
- **MySQL** (услуги, заявки, контент)
- **Vue 3** (через CDN, без сборщика)

## Быстрый старт (XAMPP)
1. Установите XAMPP и запустите `Apache` + `MySQL`.
2. Скопируйте проект в папку `htdocs`, например:
   `C:\xampp\htdocs\RA-Jiraf`
3. Импортируйте структуру БД:
   в phpMyAdmin выполните SQL из файла `db/schema.sql`.
4. Откройте сайт в браузере:
   - Главная: `http://localhost/RA-Jiraf/`
   - Услуги: `http://localhost/RA-Jiraf/services.php`
   - Админка (только по прямой ссылке): `http://localhost/RA-Jiraf/admin.php`

ТЕКУЩИЙ ЛОГИН:ПАРОЛЬ: `admin:123`


## Новые бизнес-модули
- **Регистрация и вход клиентов**: клиент создаёт аккаунт через `register.php`, входит через `login.php` и только после входа может оформить заказ.
- **Личный кабинет**: в `cabinet.php` клиент видит текущие заказы, историю заказов, статусы и может изменить своё имя.
- **Оформление заказа с расчётом**: калькулятор находится в модальном окне заказа, рассчитывает цену по услуге, количеству, площади и параметрам из админки.
- **Админский CRM-пайплайн**: заказы отображаются в `admin.php`, менеджер меняет статусы `на рассмотрении`, `в процессе`, `готово`; статус сразу виден клиенту.
- **Файлы макетов**: клиент или менеджер может прикрепить PDF/JPG/PNG/WEBP/SVG/ZIP к заказу.

Дополнительные таблицы создаются автоматически при входе в админку или обращении к API, но для чистой установки также можно импортировать полный `db/schema.sql`.
