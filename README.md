# RA-Jiraf

Стек:
- **Vue 3** (frontend, через CDN)
- **PHP** (backend API и админка)
- **MySQL** (данные услуг, заявок и редактируемого контента)

## Админка

- Доступ: `admin.php` (прямая ссылка).
- Авторизация: логин/пароль из `config/admin.php`.
- Для настройки скопируйте `config/admin.php.example` в `config/admin.php` и задайте свои данные.

Через админку можно:
- редактировать тексты на главной странице;
- добавлять, редактировать и удалять услуги;
- просматривать заявки пользователей.
