# Список и описание основных сущностей

## 1) Администратор (Admin)
Пользователь с доступом в админ-панель по логину и паролю. Может изменять контент сайта и просматривать входящие заявки.

## 2) Контент главной страницы (SiteContent)
Набор текстовых блоков главной страницы, которые редактируются без изменения кода:
- заголовок первого экрана;
- подзаголовок первого экрана;
- заголовок секции услуг;
- подзаголовок секции услуг;
- заголовок нижнего блока (CTA);
- текст нижнего блока (CTA).

## 3) Услуга (Service)
Элемент каталога услуг. Содержит:
- категорию;
- название;
- описание.

Услуги используются для отображения каталога и могут добавляться/изменяться/удаляться через админку.

## 4) Заявка пользователя (ClientRequest)
Запрос, который оставляет посетитель через форму связи на сайте. Содержит:
- имя;
- телефон;
- комментарий;
- дату создания.

Заявки сохраняются в БД и доступны для просмотра в админке.

# Что редактируется через админку

1. **Тексты главной страницы** (ключевые заголовки и описания блоков).
2. **Каталог услуг**:
   - добавление услуги;
   - редактирование услуги;
   - удаление услуги.
3. **Заявки пользователей** — только просмотр списка (без редактирования).
